# Note-sharing-app
A modern and user-friendly web application that allows users to upload, manage, and share study notes easily. This platform helps students collaborate by accessing notes anytime and anywhere.

 Features

-  Upload and share notes
-  User Authentication (Login & Signup)
-  Organize notes by subjects/categories
-  Search functionality
-  Save or bookmark notes
-  Responsive design for mobile and desktop
-  Secure storage support

 Tech Stack
 
- Python
- HTML
- CSS
- JavaScript
- React.js
- Node.js
- Express.js
- MongoDB

Future Enhancements
 - Dark Mode
 - Real-time collaboration
 - PDF Preview
 - Comment section
 - AI-generated note summaries
